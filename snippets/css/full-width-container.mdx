This tip will help you to create a full width element. It does not matter if the element is inside other element.

<Code language="css">
  {`.w-full-container {
  left: 50%;
  margin-left: -50vw;
  margin-right: -50vw;
  position: relative;
  right: 50%;
  width: 100vw;
}`}
</Code>

<Code language="xml">
  {`<section class="w-full-container">
  This is my full width element
</section>`}
</Code>
